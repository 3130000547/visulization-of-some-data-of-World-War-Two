# 二战数据可视化  
## 实验目的和要求  
1．熟悉和掌握用数据可视化的各种方法。
2，自己编写代码用d3.js实现一个可视化系统。
3，用甘特图显示个国家战争持续时间的长短，平行坐标图显示各国作为矛盾挑起者和被动响应者的次数差别，用力学图显示各国之间的矛盾关系对。
